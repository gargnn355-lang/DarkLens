# Frontend - Dark Web Onion Crawler

This is the React frontend for the Dark Web Onion Crawler project.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. The app will be available at http://localhost:5173 (or as shown in the terminal).

## Features
- Search and view crawled .onion links
- Manage sources
- Dashboard and statistics

## Requirements
- Node.js >= 18
- The backend crawler and Supabase must be running for full functionality.

## Troubleshooting
- If you see errors about missing packages, run `npm install` again.
- For API errors, check your backend and Supabase setup.
